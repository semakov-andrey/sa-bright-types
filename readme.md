# sa-bright-types

### Generating global types and type guards for starting Frontend project.

### Requirements: 

- nodejs@16.0.0

### Installation:

```bash
cd ./my-new-project
npx sa-bright-types -p src/utils
```

### Reinstallation: 

```bash
cd ./my-new-project
yarn install-types -p src/utils
```

### Update:
```bash
cd ./my-new-project
yarn update-types -p src/utils
```

#### Source:

- [react](https://github.com/facebook/react)
